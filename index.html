Differential Mobile Robot

A differential drive mobile robot was designed and implemented using a distributed ROS 2 architecture. Non-parametric system identification was performed on DC motors in MATLAB to design and tune individual velocity controllers. Differential drive kinematics and odometry were implemented to estimate the robot pose in both robot and inertial frames.

  • Technologies: ROS 2, Raspberry Pi 4, Raspberry Pi Pico (micro-ROS), DC motors with encoders, PWM motor drivers, MATLAB.
  • Key Achievement: Successfully integrated system identification, control design, embedded implementation, and ROS 2 communication into a fully functional differential mobile robot with closed-loop velocity control and real-time odometry.
